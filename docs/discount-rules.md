# Правила применения скидок

## Общие правила

- Всегда применяется **ОДНА лучшая скидка** (max%)
- Скидки **НЕ суммируются**
- Скидка на корзину и Акция конкурируют между собой

## Сравнение: Акция vs Скидка на корзину

| Критерий | Акция | Скидка на корзину |
|----------|-------|-------------------|
| Условие | Время (validFrom/validTo) | Сумма корзины (minSum) |
| Когда применяется | В даты акции | При достижении порога |
| Тип скидки | % от суммы | % от суммы |

## Алгоритм применения

```
1. Рассчитываем сумму корзины по РОЗНИЧНЫМ ценам:
   baseCartSum = sum(товар × розничная цена)

2. Находим активные акции (по времени):
   promotions = WHERE isActive = true
                AND now в validFrom..validTo
                AND applyTo IN (client.type, 'all')
   
   maxPromotionDiscount = MAX(promotions.discountPercent) ?? 0

3. Находим скидки на корзину (по сумме):
   cartDiscounts = WHERE minSum ≤ baseCartSum
                  AND applyTo IN (client.type, 'all')
                  AND isActive = true
   
   maxCartDiscount = MAX(cartDiscounts.discountPercent) ?? 0

4. Выбираем одну лучшую скидку:
   bestDiscount = MAX(maxPromotionDiscount, maxCartDiscount)

5. Применяем:
   finalSum = baseCartSum × (1 - bestDiscount/100)

6. Сравниваем: скидка vs типы цен товаров
   
   // Вариант А: товары по типам цен (опт)
   cartWithTypePrices = sum(товар × применимый тип цены)
   
   // Вариант Б: розница - скидка
   cartWithDiscount = finalSum
   
   ЕСЛИ cartWithDiscount < cartWithTypePrices
       → применяем скидку (розница - скидка)
   ИНАЧЕ
       → оставляем типы цен товаров
```

## Пример: нет скидки (не хватает суммы)

```
Корзина:
- Товар А: 5 шт × 100₽ = 500₽
- Товар Б: 5 шт × 200₽ = 1000₽
- Итого: 1500₽

Скидка от 10к: требуется 10 000₽
baseCartSum = 1500₽ (не хватает)

Итог: клиент видит 1500₽ (розничные цены)
```

## Пример: скидка выгоднее опта

```
Корзина:
- Товар А: 50 шт × 100₽ = 5000₽
- Товар Б: 50 шт × 200₽ = 10000₽
- Итого розница: 15 000₽

Скидка от 10к: 10% = 1500₽ скидка

Вариант А (типы цен):
- quantity ≥ 10 → мелкий опт 80₽/160₽
- Итого: 50×80 + 50×160 = 12 000₽

Вариант Б (скидка на корзину):
- 15 000₽ × 0.9 = 13 500₽

12 000₽ < 13 500₽ → применяем типы цен (выгоднее!)
```

## Пример: акция выгоднее

```
Корзина:
- Товар А: 5 шт × 100₽ = 500₽
- Товар Б: 5 шт × 200₽ = 1000₽
- Итого: 1500₽

Акция "Новогодняя" 10% (активна)

Вариант А (типы цен):
- Розница = 1500₽

Вариант Б (акция):
- 1500₽ × 0.9 = 1350₽

1350₽ < 1500₽ → применяем акцию!
```

## Приоритет применения

```
1. Акции (время) — берём max%
2. Скидки на корзину (по сумме) — берём max%
3. Применяем одну скидку (лучшую для клиента)
4. Сравниваем: (розница - скидка) vs (оптовые цены)
```
