# Типы цен

## Описание

Справочник типов цен — базовые цены для разных категорий клиентов.

## Сущность "Тип цены"

| Поле        | Тип      | Описание                                      |
| ----------- | -------- | --------------------------------------------- |
| id          | UUID     | Уникальный идентификатор                      |
| name        | string   | Название (Розница, Оптовая, Сотрудник и т.д.) |
| description | string   | Описание типа                                 |
| isPublic    | boolean  | Видна покупателям в карточке товара           |
| minQuantity | int      | Минимальное количество для применения         |
| createdAt   | datetime | Дата создания                                 |
| updatedAt   | datetime | Дата обновления                               |

> Активность: публичные типы (isPublic=true) участвуют в расчёте корзины

## Пример справочника

| Название        | isPublic | minQuantity | Описание                         |
| --------------- | -------- | ----------- | -------------------------------- |
| Розничная       | true     | 1           | Основная цена для клиентов       |
| Мелкий опт      | true     | 10          | Опт от 10 штук                   |
| Крупный опт     | true     | 100         | Крупный опт от 100 штук          |
| Оптовая продажа | true     | 1           | Цена для оптовых клиентов        |
| Сотрудник       | true     | 1           | Спеццена для сотрудников         |
| Закупочная      | false    | -           | Цена закупки (для расчёта маржи) |
| Маркетплейсы    | false    | -           | Средняя цена на маркетплейсах    |

## CRUD операции

- **Создание** — админ может добавить новый тип цены с названием и лимитом
- **Редактирование** — изменение названия, лимита, описания
- **Деактивация** — удалить все связи с товарами
- **Удаление** — только если тип не используется в товарах

## Валидация

| Правило                                  | Описание                     |
| ---------------------------------------- | ---------------------------- |
| minQuantity ≥ 1                          | Минимальное количество от 1  |
| minQuantity = null если isPublic = false | Внутренние типы без лимита   |
| Уникальность name                        | Название типа цены уникально |

## API эндпоинты

| Метод  | Путь                 | Описание                     |
| ------ | -------------------- | ---------------------------- |
| GET    | /api/price-types     | Список типов цен             |
| POST   | /api/price-types     | Создать тип цены             |
| GET    | /api/price-types/:id | Получить тип                 |
| PATCH  | /api/price-types/:id | Обновить тип                 |
| DELETE | /api/price-types/:id | Удалить тип цены (полностью) |
