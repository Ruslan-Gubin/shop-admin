# Товар и цены

## Сущность "Товар"

| Поле          | Тип      | Описание                        |
| ------------- | -------- | ------------------------------- |
| id            | Int      | Уникальный идентификатор        |
| name          | string   | Название товара                 |
| purchasePrice | decimal  | Закупочная цена (себестоимость) |
| prices        | relation | Коллекция цен по типам          |

## Структура цен товара

Связь через таблицу `product_prices`:

- product_id → FK products
- price_type_id → FK price_types
- price → Int

## Интерфейс добавления товара

При добавлении товара отображается таблица с колонками:

| Поле                    | Описание                  |
| ----------------------- | ------------------------- |
| Закупочная              | Себестоимость товара      |
| Розничная               | Базовая цена              |
| Мелкий опт (от 10 шт)   | Цена для мелкого опта     |
| Крупный опт (от 100 шт) | Цена для крупного опта    |
| Оптовая продажа         | Цена для оптовых клиентов |
| Сотрудник               | Спеццена для сотрудников  |

## Валидация

| Правило                  | Описание                                  |
| ------------------------ | ----------------------------------------- |
| purchasePrice ≥ 0        | Себестоимость не может быть отрицательной |
| purchasePrice ≤ retail   | Розница не ниже закупочной                |
| Все типы ≥ purchasePrice | Любая цена ≥ закупочной                   |

## Пример ошибок

- purchasePrice 50₽ → retail 40₽ ❌ Розница ниже закупочной
- purchasePrice 50₽ → smallWholesale 50₽ ⚠️ Цена равна закупочной

## API эндпоинты

| Метод | Путь                        | Описание                |
| ----- | --------------------------- | ----------------------- |
| GET   | /api/products               | Список товаров с ценами |
| POST  | /api/products               | Создать товар           |
| GET   | /api/products/:id           | Получить товар с ценами |
| PATCH | /api/products/:id           | Обновить товар          |
| POST  | /api/products/:id/auto-fill | Автозаполнить цены      |
