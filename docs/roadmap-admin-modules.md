# Roadmap: Модули админки для интернет-магазина

## Контекст проекта

**Бизнес-модель:** E-commerce с доставкой (физические товары)
**Тип магазина:** Розница / Маркетплейс
**Склады:** Мультисклад (несколько локаций)
